# 瓦器人增强 (DST Robot Pro)

为饥荒联机版（Don't Starve Together）的瓦器人（Storage Robot）和薇机人（Winona Storage Robot）添加实用增强功能。

## 功能

### 背包存储（9格容器）
- 为机器人添加 3×3 共 9 格的内置背包，右键机器人即可打开。
- 机器人拾取物品时优先存入自身背包，背包满后再存入附近箱子。
- 提供**「提取全部」**按钮，一键将机器人背包中的所有物品转移到玩家身上。

### 食物保鲜
- 存放在机器人背包中的食物腐烂速度可通过配置调节。
- 可选：永不腐烂 / 0.1x / 0.25x / 0.5x（默认）/ 正常速度。

### 自动采集
- 机器人会自动采集工作范围内的植物资源，采集后直接存入自身背包。
- 支持离屏采集（玩家不在附近时仍会工作）。
- 可采集的资源种类均可在 Mod 设置中独立开关：
  - 草丛、树枝
  - 浆果、多汁浆果
  - 蘑菇（红/绿/蓝）
  - 芦苇
  - 胡萝卜
  - 洞穴香蕉
  - 地衣
  - 石果（月球岛）
  - 猴尾草
后续添加更多支持

### 配方调整
- 为原版瓦器人添加了制作配方：齿轮 ×4 + 废铁 ×8，需要一本科技。

## 安装

将本 mod 文件夹放入饥荒联机版的 `mods` 目录下，在游戏中启用即可。

## 配置

在 Mod 设置界面中可调整：
- **保鲜程度** — 机器人背包内食物的腐烂速率倍数
- **自动采集** — 总开关及各资源类型的独立开关

## 兼容性

- 同时支持瓦器人（storage_robot）和薇机人（winona_storage_robot）。
